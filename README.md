# VoteWise AI

## Problem
Election misinformation and lack of voter awareness reduce democratic participation.

## Solution
VoteWise AI is an intelligent election assistant that helps users:

- understand voter registration
- detect misinformation
- practice voting decisions
- prepare for election day

## Features

- AI Voting Assistant
- ML Fake News Detection
- Election Scenario Simulator
- Personalized Guidance
- Multilingual Support
- Smart Intent Routing

## Tech Stack

Frontend:
- React
- TypeScript
- Tailwind

Backend:
- FastAPI
- MongoDB

AI/ML:
- Groq API
- Llama 3.1
- Logistic Regression
- TF-IDF Vectorizer

## ML Model

Custom misinformation detection model trained using TF-IDF + Logistic Regression.

## Architecture

User → Frontend → FastAPI → AI + ML Engines

## Accessibility

- multilingual UI
- mobile responsive
- simple language

## Security

- API keys stored in environment variables
- protected backend routes

## Future Scope

- deepfake detection
- regional language expansion
- election analytics
